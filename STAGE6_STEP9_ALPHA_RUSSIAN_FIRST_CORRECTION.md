## 1. Correction

The Alpha profile must NOT become an English-language UI.

The current project phase is Russian-first.

The earlier draft overcorrected Alpha separation by treating `code-like / brainrot-native / compressed` as if it meant `English-first`. That interpretation is wrong.

Correct rule:

Examples:

- `го`
- `правила`
- `репорт`
- `отмена`
- `ещё`

- `start`
- `rules`

Mostly Russian syntax with compact code-like structure.

Good direction:

- `чекаю`
- `→ {location}`
- `ещё нельзя`
- `КД`

Bad direction:

- `checking`
- `move > {location}`
- `event locked`
- `wait result`

when a natural Russian Alpha rendering exists.

Good:

- `W`
- `L 💀`
- `репорт W`
- `бро cooked`
- `аура −1000`
- `NPC тейк`
- `mid`
- `W мув`

The sentence does not need to be pure Russian, but the profile must still read as Russian-speaking rather than English-localized.

### I3_FULL_FREEDOM

Examples:

- `бро чё 💀`
- `NPC момент`
- `L тейк`

- `W move`

Even here, English density is a stylistic option, not a mandatory default.
