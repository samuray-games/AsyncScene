## 1. Direct user rule

All five generation profiles in the current Russian-first phase must represent **Russian-speaking generational culture**, not translated foreign generation stereotypes.

Target profiles:

- Boomer
- Gen X
- Millennial
- Zoomer
- Alpha

Foreign/English-language examples may be used only as:

- semantic reference;
- structural inspiration;
- evidence for the meaning of a borrowed meme/slang term;
- comparison material.

They are never automatic copy authority for Russian UI text.

## 2. Core principle

> Same generation label does not imply same cultural voice across countries.

A Russian Boomer is not an American Boomer translated into Russian.
A Russian Gen X is not a US Gen X stereotype with Cyrillic nouns.
A Russian Millennial is not a literal translation of US 90s/2000s internet culture.
A Russian Zoomer is not English TikTok copy translated word-for-word.
A Russian Alpha is not an English UI with a few Russian particles.

Every profile must be authored from the actual Russian-speaking cultural environment of that generation.

## 3. Coincidence rule

Some phrases, memes, loanwords or reaction codes may legitimately coincide with international usage.

Coincidence is allowed only when the expression is independently natural in Russian-speaking usage.

Examples of potentially shared/globalized markers:

- W / L
- NPC
- aura
- cooked
- cringe / `кринж`
- vibe / `вайб`
- skip / `скип`
- report / `репорт`

But even when the token is shared, the surrounding sentence rhythm, grammar, humour and cultural assumptions should remain Russian-native.

Do not preserve foreign syntax merely because the borrowed token is international.

## 4. No culture mixing

Forbidden failure modes:

- Russian Boomer speaking like a translated American sitcom dad;
- Gen X built from US `dude / whatever / MTV` stereotypes without Russian context;
- Millennial reduced to US `MySpace / Starbucks / avocado-toast` clichés;
- Zoomer written as literal English TikTok captions rendered in Cyrillic;
- Alpha written as English-only brainrot even when Russian-speaking peers would mix Russian and borrowed code;
- importing foreign school, workplace, money, platform or social references that do not belong naturally to Russian-speaking experience;
- combining unrelated Russian and foreign nostalgia in one profile just because both are old/young.

## 5. Russian cultural source lanes

These are broad creative lanes, not mandatory token lists.

### Boomer

Possible Russian cultural memory layers:

- Soviet and late-Soviet everyday speech;
- television/radio/newspaper-era phrasing;
- analogue bureaucracy and household idioms;
- pre-internet social norms;
- post-Soviet adaptation to digital systems.

Do not turn this into Soviet-propaganda cosplay or endless `товарищ` jokes.

### Gen X

Possible Russian cultural memory layers:

- late-Soviet childhood/youth;
- 1980s-1990s colloquial speech;
- post-Soviet rupture and street/market vocabulary;
- VHS/cassettes/early imported pop culture;
- first PCs, dial-up and early digital transition;
- practical dry sarcasm rather than social-media self-narration.

Do not reduce Gen X to gangster slang or nostalgia props.

### Millennial

Possible Russian cultural memory layers:

- dial-up/desktop internet;
- ICQ, forums, LiveJournal, early VK and similar Russian-speaking digital spaces;
- torrents, early online gaming, desktop messengers;
- RuNet meme history;
- self-irony formed during the transition from analogue childhood to digital adulthood.

Do not replace the whole profile with `превед`, ICQ nostalgia or office/therapy stereotypes.

### Zoomer

Possible Russian cultural memory layers:

- VK/Telegram/YouTube/short-form video/game-chat culture as actually used in Russian-speaking communities;
- Russianized borrowings and meme grammar;
- current gaming/streamer/social-platform language;
- emoji and reaction syntax;
- Russian sentence structure with selective international meme tokens.

Do not translate English TikTok captions literally and call that Russian Zoomer speech.

### Alpha

Possible Russian cultural memory layers:

- Russian-speaking short-form/video/game/stream culture from childhood;
- highly compressed chat language;
- visual and code-like reactions;
- global brainrot tokens only where already naturalized in Russian-speaking usage;
- Russian base grammar with selective W/L, aura, NPC, cooked, bro, etc.

Do not make English the default language of the profile.

## 6. Cultural validation per row

Before a row can become `APPROVED` or `FROZEN`, review these questions:

1. Would a Russian-speaking person of this generation plausibly say/read this naturally?
2. Is the cultural reference genuinely Russian-speaking, globally naturalized, or merely translated foreign culture?
3. If borrowed English is used, is it actually used this way in Russian-speaking communities?
4. Does the surrounding grammar sound Russian rather than translated?
5. Is the humour based on local cultural memory where appropriate?
6. Does the line accidentally import a foreign institution/platform/nostalgia marker that does not fit?
7. If a phrase is identical to foreign usage, is that because it is genuinely shared rather than lazily copied?
8. Does the line still preserve exact mechanic meaning?

Any uncertain answer keeps the row `DRAFT`.

## 7. Research rule

When authenticity is uncertain, research should be culture-specific.

Preferred evidence order for Russian copy review:

1. actual Russian-speaking usage/examples;
2. Russian language/culture sources;
3. Russian social/internet usage context;
4. foreign sources only to clarify original meaning of imported slang/memes.

A foreign dictionary definition can prove what `cooked` means, but it cannot by itself prove how a Russian-speaking Alpha should phrase the whole sentence.

## 9. Hidden-label test update

The hidden-label test must identify not only generation, but the intended **Russian-speaking version** of that generation.

FAIL examples:

- generation is obvious but culture feels Americanized;
- Alpha is recognisable only because half the UI is English;
- Millennial references foreign nostalgia alien to Russian-speaking users;
- Gen X sounds like a dubbed Hollywood character rather than Russian Gen X;
- Boomer is a generic global-old-person parody;
- Zoomer uses imported meme syntax unnaturally.

## 10. Codex boundary

Codex must not:

- translate foreign examples into Russian;
- invent cross-cultural substitutions;
- decide that an English meme is automatically valid for Russian copy;
- merge cultural lanes between generations;
- normalize Russian-specific references into generic international wording.

Codex consumes only exact user + ChatGPT approved/frozen Russian copy.
