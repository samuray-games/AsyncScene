## Жёсткие ограничения (текущая фаза)
- Ядро и экономика НЕ трогаются
- Cleanup запрещён
- Никакие расчёты, условия и состояния не меняются
- Нельзя “частично внедрять” экономику, реванш, штрафы REP за “послать”, баланс сил, NPC-правила
- Всё перечисленное — в `BACKLOG`

## Единственно допустимое в фазе
- UI/UX честность
- Убрать вводящие в заблуждение UI сигналы
- Не обещать игроку цены/награды/штрафы/трансферы/дельты
- Даже если в коде это реально происходит — UI не должен это подтверждать

## Gate от Валеры (PASS): разрешённые типы UI-действий
Разрешены ТОЛЬКО:
- `HIDE`
- `RENAME` (нейтральный текст без экономики)
- `DISABLE+LABEL` (кнопка отключена, без цен)
- `LEGACY-LABEL` (явная пометка legacy, без обещаний)
- `REMOVE-DELTA` (убрать числовые значения)

### Points
- Единственный конечный ресурс по канону
- Старт: 10 на игрока
- Только player-player трансферы
- В коде де-факто нарушено
- Исправления запрещены в текущей фазе

### REP
- Нефинитный репутационный сигнал
- Может эмититься и сгорать
- Используется как социальный индикатор
- Не является валютой

### Influence
- Изнашиваемый ресурс давления
- Определяет доступный тон
- Тон не выбирается вручную

### Кнопка “послать”
- Существует в UI
- Открывается на VL5
- Репутационный штраф за “послать” по канону не внедрён
- Внедрение запрещено в текущей фазе

### Реванш
- Зафиксирован в каноне как `BACKLOG`
- В текущей фазе запрещён к внедрению
- В коде могут быть латентные флаги; трогать нельзя (cleanup запрещён)

### NPC
- Токсик/бандит/мафиози непобедимы для обычного игрока
- Победа возможна только спецролью
- Любые изменения поведения NPC запрещены в текущей фазе
